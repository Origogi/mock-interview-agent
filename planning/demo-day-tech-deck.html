01 / Project map

Tech-Interviewer AI 전체 구조

PDF 이력서를 입력으로 받고, FastAPI가 LangGraph Agent를 실행합니다. Agent는 이력서 요약, 질문 생성, 답변 평가, 최종 리포트 생성을 담당합니다.

02 / Tech stack

Tech Stack

Frontend
React + Vite
PDF 업로드, 요약 화면, 면접 채팅, 리포트, 타임머신 UI
FrontEnd Design
Claude Design
페이지 흐름, 화면 구성, 다크 미니멀 UI, 리포트 화면 디자인 개선
Backend
FastAPI + Uvicorn
업로드, 채팅, 스트리밍, 조기 종료, 되감기 API
Agent
LangGraph + LangChain
상태 기반 면접 플로우, 노드 실행, 조건 분기
LLM
OpenAI gpt-4.1-mini
질문 생성, 답변 평가, 샘플 답변, 최종 리포트 JSON 생성
PDF
PyPDF2
업로드된 이력서 PDF에서 면접 컨텍스트로 사용할 텍스트 추출
Deploy
Railway
Frontend와 Backend를 분리된 서비스로 배포하고 환경 변수로 API 연결

04 / Interview rules

Interview Rules

Interview Policy
총 20문항, 4개 세션, 세션당 5문항으로 면접 범위를 고정합니다.
Session Order
CS Fundamentals → Framework Usage → Problem Solving → Communication 순서로 진행합니다.
Interviewer Rule
한 번에 하나의 질문만 생성하고, 이력서 요약과 이전 답변을 질문 컨텍스트로 사용합니다.
Follow-up Rule
답변의 근거, 트레이드오프, 운영 관점이 부족하면 같은 맥락에서 꼬리 질문을 생성합니다.
Evaluator Rule
정확성, 구체성, 논리성, 트레이드오프, 운영 관점을 기준으로 답변을 평가합니다.
Output Contract
질문은 문자열로, 평가는 점수/피드백으로, 리포트는 세션별 JSON 구조로 반환합니다.
05 / Key features

Key Features

PDF Resume Analysis

PDF 이력서를 업로드하면 기술 스택, 프로젝트, 경력 요약, 강점과 약점을 정리합니다.

AI Interview Questions

이력서 내용과 이전 답변을 기반으로 기술 면접 질문과 꼬리 질문을 생성합니다.

AI Sample Answers

현재 질문에 대해 Best, Good, Bad 수준의 샘플 답변을 생성해 답변 품질을 비교할 수 있습니다.

Time Machine

특정 질문 이전으로 되돌아가 다시 답변하고, 이후 평가와 리포트를 새로 만들 수 있습니다.

Final Report

세션별 점수, 강점과 약점, 개선 방향, 답변별 피드백을 최종 리포트로 보여줍니다.

Early End Report

면접을 끝까지 진행하지 않아도 일정 답변 이후에는 부분 리포트를 확인할 수 있습니다.

06 / Demo sequence

시연 순서

  1. PDF 업로드

    Resume Parser 실행, resume_summary 생성

  2. 이력서 요약 확인

    강점/약점, 기술 스택, 프로젝트 요약 확인

  3. 질문 생성

    Interviewer가 이력서 맥락 기반 질문 생성

  4. 답변 제출

    Evaluator 평가 후 다음 질문 또는 리포트로 라우팅

  5. 샘플 답변 / 타임머신

    AI 응답 생성과 checkpoint 기반 되감기 확인

  6. 최종 리포트

    Report Generator 결과와 세션별 피드백 확인
